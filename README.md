## 🚀 Student Registration System | HTML • CSS • Bootstrap • JavaScript ##

This project helped strengthen my understanding of DOM manipulation, form validation, event handling, and CRUD operations using vanilla JavaScript.

✨ Key Features:

Student registration form with Full Name, Email, Phone Number, and Course fields.

JavaScript form validation with real-time error messages.

Email validation to ensure a valid email format (including @).

Phone number validation to accept exactly 11 digits.

Prevents submission when any field is invalid.

Displays registered students in a dynamic table.

Edit existing student records.

Delete student records.

Automatically clears the form after successful submission.

Fully responsive design built with Bootstrap 5.
